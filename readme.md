# 宠物商店 主题


## 如何安装

### 1 新建 be 项目

    composer create-project be/new

### 2 安装主题（be/theme-dogfai）

    composer require be/theme-dogfai



## 部件

1. 头部 - Header
2. 商城头部标题 - ShopHeader
3. 轮播图 - Slider
4. 页面主体标题 - PageTitle
5. 横幅 - Banner
6. 横幅带内容 - BannerWithContent
7. 多个横幅 - Banners
8. 多个横幅带内容 - BannersWithContent
9. 底部 - Footer
